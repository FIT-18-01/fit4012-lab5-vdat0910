# Logs / Minh chứng

Thư mục này dùng để lưu các file minh chứng cho quá trình build, chạy chương trình và kiểm thử của bài lab.

## Ví dụ minh chứng hợp lệ

- `sample-run.log`: Log chạy chương trình mẫu
- `test-output.log`: Kết quả khi chạy lệnh `make test`
- `client-server.log`: Log giao tiếp giữa client và server
- Ảnh chụp màn hình quá trình build, test hoặc chạy chương trình
- File kết quả kiểm thử tự động (CI/CD)

- Không đưa dữ liệu thật, khóa mã hóa thật hoặc thông tin nhạy cảm vào thư mục này.
- Nên làm sạch (sanitize) dữ liệu trước khi upload lên GitHub.
- Chỉ lưu các minh chứng cần thiết phục vụ báo cáo và chấm bài.
